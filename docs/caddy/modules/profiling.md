The `profiling` app hosts the collection of push-based profiling agents with common profiling parameters acorss the Caddy instance.

[ðŸ”—](https://caddyserver.com/docs/modules/profiling#block_profile_rate)block_profile_rate

[ðŸ”—](https://caddyserver.com/docs/modules/profiling#mutex_profile_fraction)mutex_profile_fraction
